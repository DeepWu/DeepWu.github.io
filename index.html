博客更新操作指南

  1. 在博客目录下打开 git bash，输入

    hexo new “博客更新示例”

    如下图所示：

    该命令会在D:\DeepWuBlog\hexoblog\source\_posts文件夹下生成博客文档。（该博客文档使用Markdown格式编辑）

  2. 在博客文档中书写博客。

  3. 在D:\DeepWuBlog\hexoblog目录下打开git bash ，执行命令

    source up.sh

    如下图所示：

  4. 输入自己的博客网址，查看更新后的博客。 （本人博客地址：DeepWu.github.io）

补充：hexo s 可以在本地运行，输入localhost:4000可以预览结果
