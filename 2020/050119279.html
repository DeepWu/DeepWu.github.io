博客本地测试、远程更新

本地查看博客

//cmd中执行
cd hexoblog  //进入hexoblog文件夹
hexo s       //启动博客
//打开cmd中的网址，进入博客

远程更新博客

source up.sh

新建博客

进入博客目录的hexoblog目录

hexo new “新文章的名字” // 在source /_post目录下创建.md文档

发布过程

hexo clean //清理public文件夹及其内容

hexo g //生成public文件夹及其内容

hexo s // 预览

localhost:4000 // 网址预览

hexo d //发布
